# Django安裝

## 簡易流程
```shell
# 安裝
mkdir project_tsukuyomi && cd project_tsukuyomi
pipenv install django
# 啟動
pipenv shell
django-admin startproject app .
python3 manage.py runserver
# 離開
control+c
exit
```
