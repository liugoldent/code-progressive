# [CH03] 從移動平均線，抓對短中長期趨勢

## 短期移動均線
1. close > 5MA > 10MA 代表短線呈現上漲的多頭市場
  * 上漲走勢時，短線持續創高
  * 下跌時，短線反彈
  * 橫盤時，短線翻強偏多
2. close < 5MA < 10MA 代表短線呈現下跌的空頭市場
  * 呈現下跌趨勢時，短線持續創新低
  * 上漲走勢，代表漲多拉回
  * 橫盤整理時，短線轉弱偏空
3. 10MA < close < 5MA 代表橫盤整理

## 中期移動均線
* 主要以20 & 60日為參考
1. close > 20MA > 60MA，代表呈現上漲的多頭市場
  * && 金叉：買進
2. close < 20MA < 60MA，代表呈現下跌的空頭市場
  * && 死叉：賣出
3. 20MA < close < 60MA，呈現橫盤整理

## 長期移動均線
* 主要以130 & 260日做參考
1. close > 130MA > 260MA，代表長線為上漲
2. close < 130MA < 260MA，代表長線為下跌
3. 260MA < close < 130MA，代表多頭盤整
4. 260MA > close > 130MA，代表空頭盤整

## 進出場依據
* 跌破10MA，短線操作者要賣出
* 跌破60MA，中線操作者要賣出
* 跌破260MA，長線操作者要全部賣出
