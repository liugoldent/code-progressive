# [技術分析] 乖離率 - Bias

## 重點筆記
* 移動平均線代表投資人的平均成本，乖離率代表投資人平均報酬率
* 當股價在均線上
  * 乖離率為正值，代表大多數投資人獲利中，且獲利豐富，並且想獲利了結的心態越濃，賣壓較大，行情回檔機率越大
* 當股價在均線之下
  * 乖離率為負值，負值越大代表大多數投資人的虧損越嚴重，認賠殺出心裡淡薄，介入搶便宜動機越多，故行情反彈可能性越大

## 使用參數
* 長乖離率：20日&60日為基期
* 短乖離率：3-6日為基期

## 優缺點與限制
* 於多頭行情
  * 僅能指出BIAS達+20%以上有機會為賣出時機，但無法指出何時為回檔買進時機

* 於空頭行情
  * 僅能指出BIAS達-20%以下有機會為買進時機，但無法指出何時為回檔賣出時機

* 暴漲暴跌會出現極端值，但出現次數算少

* 緩漲緩跌，造成乖離率功用降低
  * 因為緩漲，會造成漲幅已大，但是正乖離率為大過高值現象出現

* 背離：
  * 當股市剛脫離空頭，乖離率初期仍會出現負值
  * 當股市從高檔反轉之下初期，乖離率初期仍會出現正值

* 故回檔時還是要使用MACD、DMI、KD做判斷買賣點

* 乖離率有助於研判短線進出場時機，但是容易失去波段行情的機會

* 於盤整行情中，乖離率功能會失效

* 建議也是使用快慢乖離率，來形成黃金交叉、死亡交叉做判斷（另外也可使用MACD做輔助）

* 若出現強者恆強、弱者恆弱，乖離率也會失效
