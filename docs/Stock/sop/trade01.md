# [SOP] 基礎知識
1. 長期趨勢（主要趨勢）：指的是一年到數年的長期走勢，又稱為空頭or多頭市場
2. 中期趨勢（次要趨勢）：指的是3weeks~3個月的走勢，主要是長期趨勢的回檔修正
  * 多頭市場：拉高後回檔
  * 空頭市場：跌深反彈
3. 短期趨勢（小趨勢）：指的是小於3週的走勢，是中趨勢進行中的調整

# 挑選個股的工具：K棒、均線、KD、RSI、MACD
* K棒：代表股價變動型態。以型態做買進or賣出
* 均線：代表趨勢。短均交叉向上代表股市向上。向下則代表偏空
* MACD：趨勢轉變。依轉紅轉黑去判斷
* KD：代表價錢位於高檔or低檔。20以下屬於超賣，可以做多。80以上屬於超買，可以做空
  * 建議還是配合成交量
* RSI：趨勢強度。20以下屬於超賣，可以做多。80以上屬於超買，可以做空
  * RSI在75以上or30以下先不進場
  * 做多建議中期指標穿越50較佳
  * 以中期RSI大於小於50作為多空分界
  * 出場原則： 
    * 做多：如果看到RSI > 75之後，預備出場，等短期RSI創下新高，回跌再出現大紅k，可以賣出

# 移動均線
* 5MA：短線操作攻擊線。
* 10MA：短線操盤線，當股價在10日均線上，代表短期走勢看漲。反之則反
* 20MA：主力控盤線，走初升段時，通常會守住不破，代表主力還是往上看。整理過後還是會走高
* 30MA：中級波段行情線，如果回檔沒跌破30MA，則中級波段依舊偏多
  * 但是若連續三天無法站上，則為空頭走勢
  * 反之連續三天站上，則為多頭走勢
* 60MA：穩健投資線
  * 對於不想冒太大風險的投資人來說，這是值得參考且風險低的均線
  * 由下往上穿越60MA時買進
  * 由上往下穿越60MA時賣出
  * 雖然風險低，但操作機會少
* 120MA or 240MA：
  * 主要是參考多、空走勢為主，很少實際用在操作上。
* 總結：
  * 大盤or個股跌破30MA，代表漲勢結束
  * 跌破60MA，代表多頭市場動搖
  * 跌破120MA，代表多頭走勢結束
  * 跌破240MA，代表空頭走勢開始

# 葛蘭碧八大法則
* 買點1：低檔出現短均穿越長均
* 買點2：短均回檔，但是沒跌破長均，然後股價再次站上所有均線，為買點2
* 買點3：短均跌破長均，然後再次向上穿越長均
* 賣點1：短均由上往下跌破均線，股價也在短均以下
* 賣點2：短均回檔上揚，但是長均仍繼續跌，並且短均無法向上穿越長均，再次跌破所有均線，加碼賣出
* 賣點3：短均穿過長均後，又再度跌落長均之下

# 均線運用類型
## 一條均線
* 主要可以使用60MA來買賣，但記得在60MA以下不做多
## 兩條均線
* 主要可以用10MA & 30MA操作（因5MA容易騙線，30MA可判斷中波段行情
## 三條均線
* 5MA + 10MA + 30MA
  * 5MA 向上穿越 10MA代表行情即將變多
  * 直到10MA穿過30MA，進場買股
## 四條均線
* 4條均線向上呈多頭排列：行情將出現大多頭走勢
* 4條均線向上呈空頭排列：行情將出現大空頭走勢
* 2短均向上，但2長均向下：跌深反彈，之後還是會尋求底部支撐
* 2短均向下，但2長均向上：漲多回檔，之後會尋求長均支撐後，繼續向上

# 移動均線操作訊號過濾
* 通常以超過均線3%以上作為明確的轉折訊號
* 或是以3天作為主要觀察期，站上3天則代表確認

# 心法
* 高檔怕震盪，寧願量縮，也不要出量（怕是出貨量）
* 低擋怕盤，因為盤久必跌，散戶沒有耐心，就會將股票賣出
* 新手怕跌，老手怕盤（沒有趨勢），高手怕不跌（因為多空都可以做）

# 股價生命週期
* 低檔盤整：主力暗中進貨，如果發現有人跟買，有可能反手殺出，逼投資者出場
* 第一主升段：突然讓股票閃亮起來，突破盤整壓力區，注意這時有可能壓回壓力區頂部，以低進高出的方式推升股價走高
  * 如果是長紅棒拉升，代表主力實力夠，不怕散戶提早賣出
* 中段整理：
  * 成交量隨整理型態量縮，直到量縮結束，盤整時期的成交量差不多時，中段結束
* 第二主升段：
  * 在不斷的鋸齒狀成交量之後，突然出現爆大量主升段
* 轉折下跌： 
  * 有可能是爆大量隔天長上影線
  * 股價在最高價時，總是回出現大的成交量，無法創新高，或是出現3根以上的長上影線
  * 天價配天量，短期股價即使回升，但最高價也不再出現
  * 股價跌破高檔區域支撐頸線，成交量隨股價買盤買氣減弱而減少
