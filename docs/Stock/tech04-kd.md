# [技術分析] 隨機指標 - KD

## 重點筆記
* 計算流程
  * 計算出九天內收盤價與最低價的差距，佔9日內最高價與9日內最低價差距之百分比
  * 再求出未成熟隨機值 - RSV
  * 再求出RSV 3日指數平滑移動平均線
  * 最後得出快速k線、慢速D線隨機指標

## 應用原則
* 超買超賣區的研判
  * for：正常理論：D值80以上為超買、20以下為超賣
  * for：台灣股市市場：D值95以上是超買、15以下是超賣
* 反轉：k線斜度趨向平緩，是警告訊號，表示有可能回軟
* 交叉點買賣依據
  * K向下突破D值，且D值在80以上出現：賣出
  * K向上突破D值，且D值在20以下（超賣區）：買進
* 另一種非交叉點交易：
  * KD於80上盤整期間，當跌破80時，去賣出
  * KD於20以下盤整，當突破20時，去買進
* 在另一種搶短線（極短線）作法：
  * K值跌破20時，賣出。等K值向上突破D時買進。
  * K值突破80時，買進。跌破D值時，賣出。
* 背離：
  * 股市創新高or新低時，K or D 值沒有再往上或往下，有可能背離出現
  * 頭部排列：（股價創新高，KD沒跟上）
    * 波段峰位下滑，但KD峰位卻墊高
    * 在第二個峰位，雖然價格沒有新高，但KD卻更高
    * 稱此為出貨型態
  * 底部背離：（股價創新低，KD沒繼續往下）
    * 大盤價格有谷底墊高，但是KD的谷底卻更低
* KD可用於短、中、長線上，但一般仍然以短線投資為主
* KD：可同時觀察日、週、月K，效果也不錯

## 參數設定
* 一般以KD-9為主
* 調整型KD指標：KD=14K value - 50 * [(H14 - L14) / (H100 - L100)]
* 快速KDJ指標
  * 意義：較為敏感的震盪指標，敏感度較現行一般KD-J為高
  * 研判： 
    * 較KD-J敏感
    * 適用任何時機
    * 超買：D>80、超賣：D<20
    * 處於超買or超賣反轉機會高
    * 當指標在超賣區（D < 20），就等黃金交叉時 or J值要<=0，為買進訊號
    * 當指標在超買區（D > 20），就等死亡交叉時 or J值大於100，為賣出訊號
  * 採順勢操作，可提高績效
  

## 另一個動向指標：NVR
* 為比KD更短期、敏感的分析工具
* 優點：KD比RSI準確率更高、NVR抓K&D值的背離
* 缺點：KD-J的交叉有時會被騙線，因此以NVR為作為波段進出，有機會獲利極高
* 研判技巧：
  * NVR極低時買進，表示否極泰來
  * 衝過0線時，表示短期多頭，可買進
  * 極高值反轉向下買出，表示物極必反
  * 跌破0時，表示短期空頭，可賣出
  * 搭配KDJ時，會更準確

## 優點
* KD頗能掌握大盤短期波動行情
* 非常適合短線進出，因為參數為9、3
* 能表現出一段行情的真正波幅
* 適合波浪理論，可以確認上升波
* 不像RSI對漲跌的比率觀測，而是趨勢&短期的綜合體
## 缺點
* 凡事以0~100表現的線一定有死角，所以超買超賣會調整為95% / 15%
* 因為做很短，所以有時容易被騙線
* 因為太短，難以判斷趨勢，是此指標最大罩門

## 使用限制
* 最好配合股票的支撐壓力區與圖型來判斷是否突破，並搭配KD交叉，才能更有效
* 股價配合KD、RSI與同期低點同時出現買進訊號，才是極佳買點
* 常出現延遲現象，雖然D在80上下不斷震盪，但股價不斷創新高，有發生這種背離時，還需要依照其他技術分析才可靠
* 交易量過小的股票不適合這個指標，適用於熱門股，準確率極高
