# [技術分析] Tower - 寶塔線

## 重點筆記
* 相較於其他指標的買進賣出互相矛盾狀況所迷惑，寶塔線提供簡單容易操作的分析方式參考
## 應用原則
* 寶塔線翻白後，股價後市會延伸一段行情。而翻黑後，也會延伸一段下降行情。
* 在高檔時，寶塔線長黑而下，應即獲利了結，將手中持股賣出
* 在低檔時，寶塔線長紅而上，則是買進時機，小翻白翻黑不必理會
* 系列收盤指標：收盤連續三天收紅，則買進。連續三黑則賣出。
* 三連發策略： 
  * 買進
    * 今天收盤價高於先前第5天收盤價＋
    * 先前第5天高於先前第10天收盤價＋
    * 先前第10天收盤價高於先前第15天收盤價
  * 賣出
    * 今天收盤價低於先前第5天收盤價
    * 先前第5天低於先前第10天收盤價＋
    * 先前第10天收盤價低於先前第15天收盤價 
## 優點
* 突破前三天最高價，為買進。跌破前三天的最低價，為賣出。
* 在多頭行情中，可長線操作，避免受到短期小幅度變動所影響提早賣出，可獲得較大波段上漲利論
* 敏感的買賣訊號，但騙線機率也高
* 可除去大漲小回或大跌小漲的買賣次數，用以驗證波浪理論不錯
* 上下波動的次數越多、時間越長，則使用寶塔線方法獲利越多
* 寶塔線採用的日期參數越短，操作上較靈活，但風險大

## 缺點
* 盤局中，小翻白or小翻黑出現頻繁，但沒有改變原先股價走勢，失誤率高，會降低獲利能力
* 暴漲暴跌時，三條陽線陰線的漲跌幅度可觀，若以寶塔線操作，會錯失機會
* 多頭轉空頭時，利用寶塔三日、四日法，不能立即反應反轉訊號
* 僅以價的表現，作為買賣進出的依據，沒有考慮到量
* 無量下跌或盤整時，常出現假買點or賣點

## 使用限制
* 配合移動平均線等配合使用，效果更佳
* 翻黑後突然翻白，要留意假突破
* 只能買到次低or賣到次高點
