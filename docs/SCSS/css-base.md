# [CSS] 選取器基本操作

## 後代-子選取器 nth-child
#### 初始值為1
### :nth-child(even)
* 選取**偶數物件**
### :nth-child(odd)
* 選取**奇數物件**
### :nth-child(2)
* 選取 第二個物件
### :nth-child(3n+1)
* n從0開始的話，則為1、4、7...
