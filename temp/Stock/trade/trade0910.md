# [CH04] 按照趨勢分析抓買點

## 基本上分三種
1. 多頭市場（偏多操作，勝率71%），拉回買點
2. 空頭市場（偏空操作，勝率70%），反彈買點
3. 盤整市場（多空操作，勝率25%），拉回買點

## 心法
* 勿因利多消息而買進，勿因利空消息而賣出
* 空頭市場：急反彈五天，緩漲8~13天，然後繼續跌

# [CH05] X象限分析法
* 利用上升趨勢線與下降趨勢線，畫出X象限
* 交叉上方為多方市場，交叉下方為空方市場
* 交叉左右為盤整市場

# [CH06] 掌握多空趨勢背離特徵
## 價量背離 -> 反轉訊號
* 正常狀況：價漲量漲、價跌量跌
* 當價格持續往上，但是成交量一波比一波低，代表價量背離

# [CH07] 洞悉多空市場何時結束
# 多頭市場結束
* M頭 -> 跌破頸線＋左邊頂點高於右邊頂點
# 空頭市場結束 
* W底 -> 突破頸線+右腳高於左腳（往上趨勢）
